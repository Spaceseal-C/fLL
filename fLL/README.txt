A projekt egy forráskódot, egy exe file-t, egy README file-t és egy 5 példából álló magyar-angol forrást tartalmaz.

Ez a projekt még mindig fejlesztés alatt van.
Ha hibát találsz kérlek jelezd.

EZ A PROGRAM NEM ÁRUSÍTHATÓ!

Hogyan használd:
    A forrásfileba ";"-el elválasztva írd be a szavakat így:"mutatott_nyelv;bekért nyelv". pl.:(igaz;true)
    Indítsd el a programot.
    A program bekéri a beolvasandó file nevét.
        A file kiterjesztését NEM kell megadni.
    Majd a program addig kéri be a választ ameddig a helyes választ meg nem adod.
    A program végtelenített ciklusban fut, be kell zárni, ahoz, hogy megálljon.
